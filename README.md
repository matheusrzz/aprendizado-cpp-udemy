# Aprendizado em C++ 🚀

Este repositório foi criado para armazenar todos os meus códigos, exercícios de fixação e projetos práticos desenvolvidos durante a minha jornada de estudos na linguagem C++.

## 🎯 Objetivo
Avançar nos conceitos de lógica de programação (saindo do VisuAlg) e dominar a sintaxe, a orientação a objetos e as boas práticas de desenvolvimento em C++.

## 📚 Curso de Origem
* **Plataforma:** Udemy
* **Status:** Em andamento ⏳

## 🗂️ Estrutura do Repositório
O projeto está organizado em pastas que seguem o cronograma de seções do curso:

* [ ] `secao-01-variaveis/` — Conceitos de variáveis e tipos de dados.
* [ ] `secao-02-funcao-if/` — Estruturas condicionais (IF/ELSE).
* [ ] `secao-03-switch/` — Estruturas de escolha múltipla (Switch-Case).
* [ ] `secao-04-while/` — Laços de repetição com While.
* [ ] `secao-05-for/` — Laços de repetição com For.
* [ ] `secao-06-array-vetor/` — Estruturas de dados lineares (Vetores e Matrizes).
* [ ] `secao-7-funcoes/` — Modularização e criação de funções.
* [ ] `secao-8-enum/` — Tipos enumerados em C++.
* [ ] `secao-9-poo/` — Programação Orientada a Objetos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** C++
* **Compilador:** GCC / G++
* **Ambiente de Desenvolvimento:** VS Code / IDE / Dev C++
